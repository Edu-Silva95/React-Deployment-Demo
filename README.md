In this project I practice using Lazy Loading and deploying a static web page using Firebase Hosting.
Current live URL: https://react-deployment-demo-c77c3.web.app/

This project was built with and for the "React - The Complete Guide 2025 (incl. Next.js, Redux)" - Udemy Course.

The topics of the lectures for this project:

- Module Introduction
- Deployment Steps
- Understanding Lazy Loading
- Adding Lazy Loading
- Building the Code For Production
- Deployment Example
- Server-side Routing & Required Configuration
